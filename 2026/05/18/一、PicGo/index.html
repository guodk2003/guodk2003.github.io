一、建立 Github 仓库

二、PicGo

  1. 什么是图床

图床就是专门用来存图片、并给你可直接用的图片链接的在线服务或服务器。常见的图床有阿里云 OSS、腾讯云 COS、PicGo 等等，笔者采用 PicGo 配置。

  1. 下载 PicGo

下载 链接，根据电脑配置选择合适版本，笔者电脑设备为 windows10，采用如下版本下载。

下载后进行安装，记住安装路径。

  1. 配置 PicGo

进入 PicGo 后，将如下框住地方修改为 Github。

点击图床设置，选择 Github 设置，按如下格式设置。

  • 图床配置名：任取，方便记忆
  • 仓库名：Github 用户名/仓库名
  • 分支名：main
  • Token：在 Github 仓库配置中保存的 Token 复制过来
  • 存储路径：可设可不设，设置 img/ 方便管理
  • 自定义域名：可以加快访问图片的速度，https://cdn.jsdelivr.net/gh/[Github 用户名]/[仓库名]@main

点击确定，配置完成。
此外，还需点击 PicGo 设置的 sever，设置监听端口需为 36677。
